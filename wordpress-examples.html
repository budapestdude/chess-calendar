WordPress Calendar Integration Examples

Copy and paste these code snippets into your WordPress pages (in HTML/Code block mode)

1. Top/Special Events Page

<script>
const CALENDAR_CONFIG = {
    dataSource: 'special-events',
    title: 'Top Chess Tournaments 2025',
    subtitle: 'The most prestigious chess events of the year',
    apiBase: 'https://chess-calendar-production.up.railway.app'
};
</script>
<iframe src="https://chess-calendar-production.up.railway.app/calendar-flexible.html" 
        width="100%" height="800" frameborder="0"></iframe>
